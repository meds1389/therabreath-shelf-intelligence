# TheraBreath Premiumization Intelligence

**Decoding Premiumization with Shelf Intelligence**

An interactive competitive intelligence microsite analyzing how category leaders use language to create premiumization across product tiers — and translating those patterns into actionable recommendations for TheraBreath.

## What This Is

A single-page interactive report built for Bluedog Design's internal team, showcasing:

- **8 Universal Premiumization Patterns** identified across 126 products from 5 category leaders (Olay, Listerine, Neutrogena, Secret, TheraBreath)
- **The Premiumization Language Ladder** — Foundation → Amplifier → Apex framework
- **TheraBreath Gap Analysis** — strategic heat map of 8 capability gaps with impact scoring
- **Three-Tier Architecture** — Essentials / Complete / Clinical Solutions recommendation
- **Brand Playbooks** — signature moves from each mentor brand
- **AI Methodology** — how Bluedog's Shelf Intelligence tool powered the analysis

## Built With

- React (bundled via esbuild)
- Bluedog's proprietary Shelf Intelligence tool
- Retail field photography from Mariano's, Walgreens, and Walmart

## Deployment

Self-contained `index.html` — all dependencies, images, and code bundled inline. No external CDN dependencies.

Deployed to GitHub Pages. To update:
1. Edit the `.jsx` source in Claude
2. Rebuild `index.html` with esbuild bundle
3. Upload new `index.html` to this repo (delete old first)
4. GitHub Pages auto-redeploys in 1-2 minutes
